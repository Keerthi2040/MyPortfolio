Experimental Study on Serverless Applications

Research Overview

  • Conducted an experimental study to analyze the performance of **serverless web applications** built with **AWS Lambda** and **API Gateway** under various load conditions.
  • Implemented a simple CRUD application using Lambda, S3, and DynamoDB, then subjected it to multiple concurrent user requests to evaluate latency, scalability, and failure handling.
  • Used **CloudWatch** metrics and manual inspection to assess how performance changes with increasing invocation load and cold start frequency.
  • Findings emphasized how memory allocation, warm starts, and I/O operations affect response time and availability in serverless setups.
  • This study bridges academic inquiry with real-world infrastructure behavior and offers practical considerations for developers building with serverless architectures.
